# HackerMan

## Description:
Mess with your friends terminals in a fun but not mean way

## To Run:
  - git clone https://github.com/HMSBennett/HackerMan
  - cd HackerMan
  - ./hacked.sh
  - cd ..
  - rm -rf HackerMan
  - history (to find current number)
  - history -c <last 6 history numbers including current number>
  
  
## Commands
  - hacked
  - youbeenhacked
